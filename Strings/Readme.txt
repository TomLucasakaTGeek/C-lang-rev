Strings

String is an array of characters
It can be accessed easily than int(or other arrays), as there is no hassle of accessing it using index

Some special functions in <strings.h> header file:

strcat():
It is used to concatenate(combine) two strings
strlen():
It is used to show the length of a string
strrev():
It is used to show the reverse of a string
strcpy():
Copies one string into another
strcmp():
It is used to compare two string
